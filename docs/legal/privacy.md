# Agent Director Privacy Notice (Draft v1)

Last updated: 2026-02-17

This notice explains what data Agent Director processes and how it is handled.

## 1. Data We Process

- Trace metadata and step data required for playback, compare, and matrix workflows.
- Account/session data required for multiplayer gameplay and collaboration.
- Operational telemetry (errors, latency, aggregate usage) for reliability and balancing.

## 2. Purpose of Processing

- Deliver core product functionality.
- Maintain security and service reliability.
- Improve gameplay balance and onboarding outcomes.

## 3. Sensitive Data Handling

- Safe export and redaction controls are available and should be used before sharing.
- Internal storage and logs should avoid unnecessary raw secret retention.

## 4. Data Sharing

Data is not sold. Data may be shared with infrastructure/service providers required to operate the product.

## 5. Retention

Data is retained only as long as needed for product operation, support, and legal obligations.

## 6. Your Controls

- Request correction or deletion of account-linked data.
- Request export of your data where supported.
- Contact support via `docs/ops/support-operations.md`.

## 7. Security Measures

- Input validation, rate limiting, and redaction defaults.
- Access restrictions for sensitive operations.

## 8. Changes to This Notice

This notice may be updated; the date above indicates latest revision.

## 9. Project Notice

Agent Director is a personal R&D / passion project and is not affiliated with the maintainer's employer.

This is an operational draft and not legal advice.
